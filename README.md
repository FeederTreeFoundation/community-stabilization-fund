## Prerequisites

- Latest version of Node.js and NPM
- [Node Version Manager](https://nvm.io/)
- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)

## Running Locally

1. Run `nvm use` 
2. Run `npm install`
3. Run `npm run start`
